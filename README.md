# 股票投资学习仓库

这是一个个人学习仓库，用于在 AI 导师的引导下系统学习「股票投资/交易」，同时重点研读杰西·利弗莫尔、马克·米奈维尼等交易大师的理论，并把它们转化为适合自己的交易体系。

整体结构参考 `CFP-Study` 和 `medical-study` 仓库：每一次学习会话都有详细记录，并通过单一追踪文件维护整体进度。

## 仓库结构

```
/financial-study/
  /sessions/
    YYYY-MM-DD/
      session-notes.md
    SESSION-TEMPLATE.md

  /progress/
    financial-study-tracker.md

  FINANCIAL.md
  README.md
```

- `FINANCIAL.md`：给 AI 导师看的说明文档（角色、教学风格、学习域划分、记录规则）。
- `sessions/`：每天一个子目录，用来保存详细的学习会话记录。
- `progress/financial-study-tracker.md`：单一「学习进度真相源」，用来统筹管理股票投资学习状态。

## 使用方式

1. 每次学习当天，在 `/financial-study/sessions/YYYY-MM-DD/` 下创建 `session-notes.md`，内容可参考 `SESSION-TEMPLATE.md`。
2. 学习过程中，与 AI 沟通以下主题：
   - 股票和市场基础概念；  
   - 风险管理与资金管理；  
   - 交易心理与行为偏差；  
   - 技术分析与趋势交易基础；  
   - 杰西·利弗莫尔的方法论（原则、市场阶段、加减仓规则等）；  
   - 马克·米奈维尼的成长股交易框架（选股、买点、仓位与止损等）；  
   - 如何把大师理论整合到自己的交易系统和复盘流程中。  
3. 学习结束后，更新 `progress/financial-study-tracker.md`：
   - 本次覆盖的主题及掌握程度；  
   - 暴露出的知识盲区或典型错误模式；  
   - 下一阶段的学习与实践计划（例如：模拟盘练习、特定形态的案例收集等）。  
4. 重复这个闭环：学习 → 记录会话 → 更新进度 → 复盘与规划下一步。

## 重要提醒

- 本仓库中的所有内容仅用于学习与自我提升，**不构成任何形式的投资建议或收益承诺**；  
- AI 导师应避免对具体个股给出「买/卖」指令，而是帮助你：
  - 更清晰地理解市场与交易逻辑；  
  - 更系统地整理大师们的方法论；  
  - 更有条理地建设和迭代自己的交易系统。
